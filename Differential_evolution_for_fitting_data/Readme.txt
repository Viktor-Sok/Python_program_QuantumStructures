Для нахождения параметров фиттинга экспериментальных данных флуоресцентного распада молекулы используется метод дифференциальной эволюции для нахождения минимума целевой функции. В программе реализована работа с файловой системой для удобства сортировки и обработки  полученных  числовых данных. В разных папках находятся различный функционал, который был использован в главной программе в папке CycleTestFitting. Также в одной из папок имеется генератор данных иметирующих экспериментальные с заданным вероятностным распределением, для тестирования главной программы.
